# User and Policy Management Dashboard

A responsive and user-friendly web application for managing users and policies. It provides a clean dashboard interface to view, add, edit, and delete user and policy data.

## Features

- **Dashboard Overview:** A summary of total users and policies.
- **User Management:**
  - View a paginated list of users.
  - Search for users by name, email, or role.
  - Add, edit, and delete users through a modal interface.
- **Policy Management:**
  - View a paginated list of policies.
  - Filter policies by status (Active, Expired, Pending).
  - Add, edit, and delete policies through a modal interface.
- **Responsive Design:** The application is fully responsive and works on all screen sizes.
- **Modern UI:** A clean and attractive user interface built with React and Tailwind CSS.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
